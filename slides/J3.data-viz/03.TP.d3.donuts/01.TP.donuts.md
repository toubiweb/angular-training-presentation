## TP: Donuts charts

Basculez sur la branche 3.1:

``` bash
    // fetch modifications from server
    git fetch --all
    // switch to dedicated branch (local modifications will be lost)
	git checkout -f tp-3.1-donuts

```

Intégrez de façon statique l'exemple de donuts charts du site dans la directive app/components/chart/donut-chart:
- http://nvd3-community.github.io/nvd3/examples/site.html
- https://github.com/nvd3-community/nvd3/blob/gh-pages/examples/donutChart.html
