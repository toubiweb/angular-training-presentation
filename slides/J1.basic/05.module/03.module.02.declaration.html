Déclaration d'un module

Déclaration du module (profile.js) :


    angular.module('tw.practice.profile', []);

Injection du module dans l'application (app.js):


angular.module('tw.practice', ['tw.practice.profile']);

Le contrôleur est désormais déclaré dans le module (profile.user-edit.ctrl.js):


angular
    .module('tw.practice.profile')
    .controller('TpaProfileUserEditController', TpaProfileUserEditController);
